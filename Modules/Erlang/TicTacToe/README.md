# Overview

The TicTacToe game developed with Erlang, shows messages from participants, all this through the terminal.


[Software Demo Video](https://youtu.be/wUxWzTbFnrs)

# Development Environment

First install erlang, to complement this a new Path was added through the advanced system configuration, cd, erlc and erl commands are used to access, start the project and use the game functions.
